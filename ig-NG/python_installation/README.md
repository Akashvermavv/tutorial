Python sitere na ngwụsị 1980s ma ebum n’uche ya bu ka ndi mmadu nwe ike i guru ya (obughi nani igwe!). Nke a bụ ya mere na ọ dị mfe karịa n'asụsụ mmemme ndị ọzọ, mana echegbula - Python dịkwa ike n'ezie!
